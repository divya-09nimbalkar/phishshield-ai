🔐 AI Phishing Detection System

An intelligent cybersecurity solution that detects phishing websites using Machine Learning, a browser extension, and a real-time monitoring dashboard.

📌 Project Overview

The AI Phishing Detection System is designed to identify malicious and phishing URLs in real time. It integrates a trained ML model with a backend API, a browser extension, and a visualization dashboard to provide end-to-end protection.

🚀 Key Features
🔍 Detects phishing websites using Machine Learning
🌐 Browser Extension for real-time URL scanning
📊 Interactive Dashboard for monitoring and analytics
⚡ Fast API backend for prediction
🧠 Feature extraction from URLs
📁 Structured dataset and preprocessing pipeline
